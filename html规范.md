
#html规范

## 1 使用html5 doctype
`<!DOCTYPE html>`

## 2 页面使用utf-8
```
<meta charset="UTF-8">
```
pc专题使用gb2312编码，使用fit工具导出时勾选gb2312

## 3 使用最新浏览器
```
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />
<meta name="renderer" content="webkit" />
```
